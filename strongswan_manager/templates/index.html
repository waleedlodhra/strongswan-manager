Welcome to StrongSwan Manager — Client Edition

A complete management interface for the strongSwan VPN solution. Configure and establish IPsec connections through a user-friendly GUI.

Getting started

Welcome to StrongSwan Manager — Server Edition

A complete management interface for the strongSwan VPN solution. Manage server-side IPsec tunnels, remote access VPNs, and site-to-site connections.

Tip: Use the button in the top-right to switch between Server and Client mode.

Getting started

  • Upload certificates in the Certificates section.
  • Create site-to-site or remote access connections in the Server Connections section.
  • Manage EAP Secrets for user authentication in the EAP Secrets section.
  • Create IP address pools in the Pools section.
